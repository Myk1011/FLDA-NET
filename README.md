# FLDA-NET
A full-level domain adaptation network for cross-domain land cover classification of remote sensing images

## Requirements
- Python 3.9
- Pytorch >=1.10.1
- CUDA 11.3 or higher

## Datasets
- ISPRS Vaihingen dataset
- ISPRS Potsdam dataset

## Usage
- Train
python train.py
- Test
python evalute.py
